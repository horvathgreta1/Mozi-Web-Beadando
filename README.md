# Mozi-Web-Beadando

Ez a repository a Webprogramozás 1 tantárgy beadandó projektjének forráskódját tartalmazza. A feladat egy komplex, többrétegű webalkalmazás elkészítése volt, amely egy mozi adatbázisát (filmek, előadások) dolgozza fel.

🛠️ Használt technológiák

A projekt során a frontend és backend fejlesztés modern eszközeit alkalmaztuk:

- Frontend: HTML5, CSS3, natív JavaScript (Vanilla JS & OOJS)
- Keretrendszer: React (Vite környezetben, Babel/standalone nélkül)
- API Kommunikáció: Fetch API, Axios
- Backend & Adatbázis: PHP, MySQL (Nethely szerveren futtatva)

✨ Főbb funkciók

A weboldal a következő modulokból épül fel:

1. Alap design: Reszponzív HTML/CSS felület és menürendszer.
2. Kliensoldali CRUD: Helyi memóriában tárolt adatok kezelése JavaScript és React segítségével.
3. Szerveroldali CRUD: Valós adatbázis-műveletek (Create, Read, Update, Delete) PHP végpontokon keresztül (Fetch API és Axios implementációval).
3. OOJS: Objektumorientált JavaScript grafikus alkalmazás (osztályok, öröklődés).
4. SPA (Single Page Application): React alapú, oldalfrissítés nélküli interaktív mini-alkalmazások/játékok.

A projekt az egyetemi követelményeknek megfelelően oktatási céllal készült.
